# Website Brief — Analyst 3D Premium Platform

## 1. Overview

Build a premium, cinematic, 3D animated website for an analyst learning platform.

The website must feel like:
- an advanced AI research lab
- a futuristic data environment
- a holographic analytics system

This is NOT a typical SaaS website.
It must feel immersive, intelligent, and high-end.

Reference inspiration:
https://le-lab.io (only for feel, not copy)

---

## 2. Core Objective

Create a visually stunning, production-ready platform that:

- attracts learners and recruiters
- demonstrates real-world analyst workflows
- enables skill-based hiring
- feels futuristic and premium

Main message:
**Learn → Practice → Publish → Get Hired**

---

## 4. Design Direction

The design must feel:

- cinematic
- ultra-premium
- futuristic
- structured
- intelligent
- immersive

Avoid:
- generic SaaS look
- cartoonish UI
- clutter
- excessive animations

---

## 5. Visual Style

Color system:
- background: dark charcoal / midnight blue
- primary accent: electric cyan
- secondary tones: soft blue, cool steel, indigo glow

UI elements:
- glass panels
- soft glow
- thin borders
- layered depth
- floating UI cards
- reflections

---

## 6. Motion System

Motion must be:

- smooth
- slow
- system-like
- controlled

Include:
- holographic reveals
- scan effects
- depth-based scrolling
- floating elements
- radial expansion

Avoid:
- bouncy animation
- over-animation
- gimmicks

---

## 7. 3D / Depth Strategy

The website should feel 3D using:

- layered UI
- perspective grids
- parallax
- shadows and glow
- floating components

Optional:
- limited use of 3D (React Three Fiber) in hero

Avoid heavy 3D everywhere.
