# AGENTS.md

Project goal:
Build a premium desktop-first 3D animated website for an analyst learning platform.

Tech stack:
- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion

Rules:
- Keep layout desktop-first
- Use cinematic premium design
- Use dark charcoal / midnight blue base
- Use electric cyan accent
- Prefer layered depth over heavy 3D
- Keep motion smooth and restrained
- Avoid generic SaaS styling
- Reuse components
- Keep performance strong
- Preserve accessibility
- Use semantic HTML
